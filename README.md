## Description

Take your travel log offline wherever you are

### Features

* Support Day Night
* Custom google map to interact with

### Coming Features

* Save images on remote servor

### Technologies used
* Kotlin
* JetPack navigation
* AndroidX
* Room database
* Single Activity
* Material design
